# Visitors_Tracker App

- Tracks the number of Visitors & Stating each Visitors' Purpose of Visit.
- Application used in a Hospital Setting.

## Tools Used:
- Ruby On Rails (4.2.3)
- AngularJS (1.4.8)
- Bootstrap (3.3.5)
- Jasmine (used the Gem Version)
